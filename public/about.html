About FDA Double Check

A web application built to help users stay informed about FDA product safety.

What We Do

FDA Double Check pulls real-time data from the openFDA database to help you:

  • Look up drug ingredients, warnings, and purposes
  • Browse the latest FDA food recall announcements
  • Search reported adverse events and side effects

Data Source

All data is sourced from the openFDA API, a public API maintained by the U.S. Food and Drug Administration.

Built With

This project was built using Node.js, Express, Supabase, and the openFDA API.
